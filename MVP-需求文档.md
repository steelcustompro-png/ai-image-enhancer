# AI Image Enhancer — MVP 需求文档

## 一、项目概述

### 产品名称
AI Image Enhancer（AI 图片增强器）

### 目标关键词
- 主词：ai image enhancer, ai image upscaler
- 长尾词：enhance image quality online, upscale image free, ai photo enhancer

### 产品定位
一款在线 AI 图片增强/放大工具，帮助用户将低分辨率、模糊的图片变清晰、变高清。面向全球用户，支持中英文。

### 目标用户
- 电商卖家：需要高清产品图
- 设计师/摄影师：修复老照片、提升图片质量
- 社交媒体创作者：提升发布图片的清晰度
- 普通用户：修复模糊的个人照片

### 竞品参考
| 竞品 | 特点 | 定价 |
|------|------|------|
| Upscale.media | 支持2x/4x/8x放大，免费有限制 | 免费+付费 |
| Let's Enhance | AI增强+放大，质量好 | $9/月起 |
| iLoveIMG | 图片工具集合，含放大功能 | 免费+付费 |
| Bigjpg | 专注动漫图片放大 | 免费+付费 |

## 二、技术方案

### 技术栈
- 前端：Next.js + Tailwind CSS
- 部署：Cloudflare Pages
- 域名：Cloudflare 管理
- 数据库：Cloudflare D1（SQLite，存储用户信息和用量）

### AI 图片处理 API 选择
推荐使用 **Replicate API**（https://replicate.com）：
- 提供 Real-ESRGAN 模型（图片放大/增强的业界标准）
- 按次计费，约 $0.005/次，成本可控
- 支持 2x/4x 放大
- 注册即送免费额度

备选方案：
- Deep AI Image Enhancer API（有免费额度）
- TensorPix API

### 存储方案
- 不做服务端存储
- 图片处理完直接返回给用户下载
- 用内存/临时存储处理，降低成本

## 三、MVP 功能清单

### 核心功能（必做）
1. **图片上传** — 支持拖拽上传和点击选择，支持 JPG/PNG/WebP 格式
2. **AI 放大** — 支持 2x 和 4x 两种放大倍数
3. **效果预览** — 处理前后对比展示（左右滑动对比）
4. **一键下载** — 处理完成后直接下载高清图片
5. **中英文切换** — 自动检测浏览器语言 + 手动切换

### 用户体系（必做）
1. **Google OAuth 登录**
2. **用量控制**：
   - 未登录用户：每天 1 次免费
   - 登录用户：每天 3 次免费
   - 付费用户：按积分包购买（10次/$2.99，50次/$9.99，200次/$29.99）
3. **个人中心** — 查看剩余次数、历史记录

### 支付系统（必做）
1. **PayPal 支付** — 一次性积分包购买
2. **定价页面** — 展示免费/付费方案对比

### SEO 相关（必做）
1. **SEO 友好的页面结构** — 标题、描述、H1-H6、alt 标签
2. **Sitemap.xml** — 自动生成
3. **Robots.txt** — 正确配置
4. **Canonical 标签** — 避免重复内容
5. **页面速度优化** — 图片懒加载、代码压缩

## 四、页面规划

### 首页（核心功能页）

- 积分包购买按钮
- FAQ

### 个人中心
- 用户信息
- 剩余次数
- 购买记录

## 六、成本预估

| 项目 | 费用 |
|------|------|
| 域名（GoDaddy） | ~$10/年 |
| 服务器（腾讯云） | ¥199/年（已有） |
| Cloudflare | 免费 |
| Replicate API | ~$0.005/次，初期成本极低 |
| PayPal 手续费 | 0.5% + 6% 增值税 |

### 盈利模型
- 免费用户引流 → 付费转化
- 积分包：10次/$2.99，50次/$9.99，200次/$29.99
- 目标：月活 1000 用户，5% 转化率 = 50 付费用户
- 预估月收入：$500-$1500

## 七、风险与应对

| 风险 | 应对方案 |
|------|---------|
| API 成本超预期 | 严格控制免费额度，优先转化付费 |
| 图片处理速度慢 | 添加进度条，优化用户等待体验 |
| SEO 排名慢 | 持续发外链，保持耐心（3-6个月） |
| 竞品太强 | 差异化：更简洁的体验 + 更低的价格 |
